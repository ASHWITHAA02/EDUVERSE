## ‚úÖ Quick Reference

### Essential Commands

**Backend**
```powershell
cd eduverse_backend
.\venv\Scripts\Activate.ps1
uvicorn main:app --reload
```

**Frontend**
```powershell
cd eduverse_frontend
```

